# 🚗 NYC Motor Collisions Dashboard (Power BI)

## 🧾 Overview
This project analyzes **motor vehicle collisions in New York City**, focusing on accident patterns, contributing factors, and severity trends.

Using **Power BI**, the dataset was cleaned, modeled, and transformed into an interactive dashboard to uncover insights that can support **road safety improvements and policy decisions**.

---

## 🎯 Objectives
- Analyze overall **collision trends over time**
- Identify **high-risk locations and boroughs**
- Understand key **contributing factors** to accidents
- Evaluate **injuries and fatalities distribution**
- Detect **patterns across time (hour, day, season)**

---

## 🗂️ Dataset
The dataset contains detailed records of motor vehicle collisions, including:
- Date and time of incidents  
- Location (borough, street, coordinates)  
- Number of injuries and fatalities  
- Contributing factors (e.g., speeding, distraction)  
- Vehicle types involved  

---

## ⚙️ Data Modeling
Data preparation and modeling steps include:
- Cleaning missing and inconsistent location data  
- Standardizing contributing factor categories  
- Creating date/time hierarchies for trend analysis  
- Developing **DAX measures** for KPIs (Total Collisions, Injuries, Fatalities etc)  
- Building relationships where necessary for dimensional analysis  

---

## 📈 Dashboard Features
- 🚦 Total Collisions, Injuries & Fatalities Overview  
- 🗺️ Borough-Level & Location Analysis  
- ⏰ Time-Based Trends (Hourly / Daily / Monthly)  
- ⚠️ Contributing Factors Breakdown  
- 🚘 Vehicle Type Analysis  
- 🔍 Interactive filters for deep exploration  

---

## 🧠 Key Insights
- A collision is bound to happen at ~7minutes interval from another one with an average of 200 collisions per days
- The fatality rate is below **0.5%** with a total of 49K people injured
- Certain boroughs consistently report **higher collision volumes**  
- Peak accident periods occur during **specific hours of the day**  
- There are more **male** and **licensed driver** involved in collisions

---

## 🛠️ Tools Used
- **Power BI** (Data modeling & visualization)  
- **DAX** (Measures & calculations)  
- **Power Query** (Data transformation & cleaning)
