# 📊 Superstore Sales Dashboard (Power BI)

## 🧾 Overview
This project explores the **Superstore dataset**, where data was cleaned, modeled, and visualized using **Excel and Power BI** to generate actionable business insights.

The goal is to give an overview of key metrics, sales , profitability, orders, and product performance through an interactive dashboard.

---

## 🎯 Objectives
- Analyze overall **sales and profit performance**
- Identify **top-performing product categories and customer segments**
- Track **regional and segment-level trends**
- Evaluate **profit drivers**
- Highlight ** top customers and products**

---

## 🗂️ Dataset
The dataset used is the popular **Superstore dataset**, which includes:
- Orders
- Customers
- Products
- Sales & Profit metrics
- Geographic information

---

## ⚙️ Data Modeling
Data was transformed and modeled in Power BI:
- Created relationships between fact and dimension tables
- Built calculated columns and measures using **DAX**
- Implemented date hierarchies for time-based analysis

---

## 🛠️ Tools Used
- **Power BI** (Data modeling & visualization)
- **DAX** (Measures & calculations)
- **Power Query** (Data transformation)
